# Skill Usage Dashboard

Use this Skill when the user wants to inspect local Skills usage, open the Skills dashboard, diagnose hook status, or manage Skill enablement across Claude Code and Codex.

Expected future commands:

```bash
skill-meter open
skill-meter doctor
skill-meter scan
skill-meter import
skill-meter summary
```

`skill-meter hook ingest`, `import`, `summary`, `scan`, and `doctor` are implemented for the MVP data link. `open` is still reserved for a future dashboard launcher.
